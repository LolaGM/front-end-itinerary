Table Padding and Spacing

HTML tables can adjust the padding inside the cells, and also the space between the cells.

HTML Table - Cell Padding

Cell padding is the space between the cell edges and the cell content. By default the padding is set to 0. To add padding on table cells, use the CSS padding property.

Table Styling

HTML Table - Zebra Stripes

If you add a background color on every other table row, you will get a nice zebra stripes effect:

tr:nth-child(even) { background-color: #D6EEEE; }

HTML Table - Vertical Zebra Stripes

To make vertical zebra stripes, style every other column, instead of every other row.Set the :nth-child(even) for table data elements like this:

td:nth-child(even), th:nth-child(even) { background-color: #D6EEEE;}
